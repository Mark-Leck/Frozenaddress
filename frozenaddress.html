Why should I use a Bulk Wallet to accept Frozencoins on my website?
The traditional approach to accepting Frozencoins on your website requires that you install the official Frozencoin client daemon ("Frozencoind"). Many website hosting packages don't support installing the Frozencoin daemon. Also, running the Frozencoin daemon on your web server means your private keys are hosted on the server and could get stolen if your web server is hacked. When using a Bulk Wallet you can upload only the Frozencoin addresses and not the private keys to your web server. Then you don't have to worry about your Frozencoin wallet being stolen if your web server is hacked.
How do I use a Bulk Wallet to accept Frozencoins on my website?
  1. Use the Bulk Wallet tab to pre-generate a large number of Frozencoin addresses (10,000+). Copy and paste the generated comma separated values (CSV) list to a secure text file on your computer. Backup the file you just created to a secure location.
  2. Import the Frozencoin addresses into a database table on your web server. (Don't put the wallet/private keys on your web server, otherwise you risk hackers stealing your coins. Just the Frozencoin addresses as they will be shown to customers.)
  3. Provide an option on your website's shopping cart for your customer to pay in Frozencoin. When the customer chooses to pay in Frozencoin you will then display one of the addresses from your database to the customer as his "payment address" and save it with his shopping cart order.
  4. You now need to be notified when the payment arrives. Google "Frozencoin payment notification" and subscribe to at least one Frozencoine payment notification service. There are various services that will notify you via Web Services, API, SMS, Email, etc. Once you receive this notification, which could be programmatically automated, you can process the customer's order. To manually check if a payment has arrived you can use Block Explorer. Replace THEADDRESSGOESHERE with the Frozencoin address you are checking. It could take between 10 minutes to one hour for the transaction to be confirmed.
    http://explorer.Frozencoin.net/address/THEADDRESSGOESHERE

    Unconfirmed transactions can be viewed at: http://explorer.Frozencoin.net/
    You should see the transaction there within 30 seconds.
  5. Frozencoins will safely pile up on the block chain. Use the original wallet file you generated in step 1 to spend them.
